# ITIL 4 Preparation App - Technical Specs

## Requirements
- **Runtime**: Node.js 18+
- **Browser Compatibility**: Modern browsers (Chrome, Firefox, Safari, Edge).

## Dependencies
- `react`: ^19.0.0
- `vite`: ^6.0.5
- `lucide-react`: ^0.469.0
- `tailwindcss`: ^4.0.0

## Build & Deployment
- `npm run dev`: Start local development server.
- `npm run build`: Generate production-ready assets in `dist/`.
- `npm run preview`: Locally serve the production build.

## API & Limits
- This application currently operates purely on client-side logic without external backend dependencies, ensuring high performance and availability.
