# ITIL 4 Preparation App - Blueprint

## Project Goal
A web-based preparation tool for the ITIL 4 Foundation exam, supporting multiple languages (Russian, German) and featuring an interactive quiz runner, glossary, and exam simulations.

## Core Features
1. **Multi-language Support**: Seamless switching between Russian and German translations.
2. **Quiz Engine**: Interactive exam simulations with immediate feedback and scoring.
3. **Glossary & Search**: Centralized repository of ITIL 4 terms with global search functionality.
4. **Study Modules**: Structured content categorized by ITIL 4 core concepts.

## Tech Stack
- **Framework**: React 19 + Vite 6
- **Styling**: Tailwind CSS 4
- **Icons**: Lucide React
- **State Management**: React Hooks (useState, useEffect) and Context API for global state (translations, search).
- **Hosting**: Target platforms (Vercel/Netlify/GitHub Pages).
